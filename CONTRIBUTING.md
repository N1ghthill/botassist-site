# Contribuindo

Obrigado por considerar contribuir com o **BotAssist Site**.

Este repositório é **somente o site** (landing page). O aplicativo desktop fica em:

- `https://github.com/N1ghthill/botassist-whatsapp`

## Como contribuir

- Abra uma issue explicando o problema/ideia (principalmente para mudanças maiores).
- Faça um fork e crie uma branch a partir do `main`.
- Rode localmente:
  - `npm install`
  - `npm run lint`
  - `npm run build`
- Abra um Pull Request descrevendo o que mudou e como testar.

## Objetivo do projeto

Colaborar com o desenvolvimento de **software livre**, **gratuito** e de **qualidade**.
