# Case - BotAssist Site

## Problema
O produto precisava de um canal público para explicar proposta de valor, reduzir dúvidas e direcionar downloads corretos por sistema operacional.

## Solução
Landing page em Next.js com integração aos releases do GitHub do app principal, FAQ e materiais de suporte.

## Tech Stack
- Next.js
- React
- Tailwind CSS
- GitHub Releases API

## Status
- `production`: site oficial online com atualização contínua.
